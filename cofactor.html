Cofactor Matrix

The co-factor matrix is formed with the co-factors of the elements of the given matrix. The co-factor of an element of the matrix is equal to the product of the minor of the element and -1 to the power of the positional value of the element. The co-factor matrix is useful to find the adjoint of the matrix and the inverse of the given matrix. Here we shall learn how to find the co-factor matrix and the applications of the co-factor matrix.
